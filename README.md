# Student Career Recommendation System
## Project Overview
An interactive Tableau dashboard that analyzes student education, skills, aptitude, certifications, and career interests to provide insights into career readiness and recommended career paths.

## Dataset used : -<a href="https://github.com/aarti-ashok-dorale/Data-Analysis-Dashboard-TABLEAU/blob/main/Dataset.xlsx">Dataset</a>
## Business Questions
•	Which streams have the highest number of students? 
•	How do internships vary across education levels? 
•	Does certification impact aptitude scores? 
•	How do skills and interests influence career recommendations? 
•	Which careers are recommended most frequently? 

## Key KPIs
•	Total Students: 243 
•	Average CGPA: 7.8 
•	Average Aptitude Score: 69.7 
•	Average Certifications: 4 
•	Average Communication Skill: 5.3

## Process
1.	Collected and cleaned student data. 
2.	Performed data transformation in Tableau. 
3.	Created interactive dashboards: 
  a.Student Overview 
  b.Students Skill Analysis 
  c.Career Recommendation Analysis 
4.	Added filters and visualizations for deeper insights. 

## Project Insights
•	Mechanical and IT streams have the highest student count. 
•	Students with more certifications tend to have higher aptitude scores. 
•	Advanced programming skills indicate better career readiness. 
•	Data Analyst, UI/UX Designer, and Financial Analyst are among the top recommended careers. 

## Conclusion
The dashboard helps identify relationships between student performance, skills, and career interests, enabling data-driven career guidance and better decision-making.
